• Нормальное выполнение прерывает парсинг html страницы для загрузки и выполнения скрипта
  • async асинхронно загружает скрипт и выполняет его после загрузки, то есть парсинг html прерывается после завершения загрузки скрипта и возобновляется после загрузки скрипта. Доступен для внешних файлов
  • defer Стартует асинхронную загрузку без прерывания парсинга html но выполняет скрипт уже после полной загрузки страницы
